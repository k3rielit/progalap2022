[⭐] 4. Main (1 pont)

Hozd létre a main függvényt a main.c fájlban! A main függvényben ne történjen semmi, csak jelezze, hogy a program rendben lefutott!
A main függvény egy egész számmal tér vissza. Ha ez 0, akkor azt jelezzük az operációs rendszer felé, hogy minden rendben történt. Ha nem 0, akkor valami probléma történt a futás során. Visszatérni egy függvényből a return utasítással lehet.
